# dio-lab-conversando-por-voz-com-chatgpt
Projeto de IA conversacional por voz: integra Whisper, ChatGPT API e gTTS para criar um assistente de suporte técnico com entrada e saída em áudio. Executado via Google Colab.

# Projeto DIO - Integração com OpenAI

## 📌 Descrição
Projeto que utiliza a API da OpenAI para gerar respostas a partir de uma transcrição de áudio.

## 🚀 Tecnologias
- Python
- OpenAI API
- Google Colab

## ⚠️ Observação
Este projeto utiliza mock da API devido à limitação de créditos.

## ▶️ Como executar
1. Defina a variável de ambiente OPENAI_API_KEY
2. Execute o notebook
